# AgentDecompile

## Overview
AgentDecompile is a Python project designed to facilitate the decompilation and analysis of binary files. It provides a command-line interface for users to interact with the decompilation process and offers various utilities for data processing and analysis.

## Installation
To install the required dependencies, run the following command:

```
pip install -r requirements.txt
```

## Usage
To use the AgentDecompile tool, you can run the command line interface as follows:

```
python -m agentdecompile.cli [options]
```

### Options
- `--input <file>`: Specify the input binary file to decompile.
- `--output <directory>`: Specify the output directory for the decompiled files.
- `--analyze`: Run the analysis on the decompiled data.
