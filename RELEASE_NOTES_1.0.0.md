# AgentDecompile 1.0.0

AgentDecompile 1.0.0 is the first stable release of the MCP server and CLI for Ghidra/PyGhidra workflows.

## Highlights

- Stable `1.0.0` release baseline for CLI and MCP server workflows.
- Unified tool and parameter normalization behavior across provider dispatch.
- Consolidated tool surface for symbols, functions, memory/data, control flow, annotations, and project operations.
- Improved release/documentation consistency for tag and container version formatting.

## Container Images

- `docker.io/bolabaden/agentdecompile-aio:1.0.0`
- `docker.io/bolabaden/agentdecompile-mcp:1.0.0`

## Upgrade Notes

- Release tags and release references use plain semver (for example: `1.0.0`).
- Automation, contributor guidance, and image publishing now follow this same format.
